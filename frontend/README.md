# AuraMail Frontend

Це React-версія фронтенду для AuraMail.

## Встановлення

1. Перейдіть у папку `frontend`:
   ```bash
   cd frontend
   ```

2. Встановіть залежності:
   ```bash
   npm install
   ```

3. Запустіть сервер розробки:
   ```bash
   npm run dev
   ```

Додаток буде доступний за адресою: http://localhost:5173

## Структура

Frontend використовує Vite + React + TailwindCSS.
Ваш Flask сервер повинен бути запущений на порту 5000 для роботи API.
