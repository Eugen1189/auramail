# Gmail API Label Colors - Valid Palette

## Важливо

Gmail API має **обмежений набір дозволених кольорів** для міток. Не всі HEX коди працюють!

## Дозволені кольори Gmail API

Згідно з документацією Gmail API, дозволені кольори міток:

1. **Червоний (Red):** `#F28B82` або `#EA4335`
2. **Помаранчевий (Orange):** `#FBBC04` або `#FF9500`
3. **Жовтий (Yellow):** `#FFF475` або `#FFC107`
4. **Зелений (Green):** `#CCFF90` або `#34A853`
5. **Блакитний (Cyan):** `#A7FFEB` або `#17A2B8`
6. **Синій (Blue):** `#CBF0F8` або `#4285F4`
7. **Фіолетовий (Purple):** `#D7AEFB` або `#9C27B0`
8. **Рожевий (Pink):** `#FDCFE8` або `#E91E63`
9. **Сірий (Gray):** `#E8EAED` або `#9AA0A6`

## Рекомендації

Використовуйте **світлі відтінки** для кращої читабельності тексту білим кольором.

## Альтернативний підхід

Якщо Gmail API відхиляє конкретні HEX коди, можна:
1. Не вказувати колір (Gmail призначить колір за замовчуванням)
2. Використовувати тільки затверджені значення з документації
